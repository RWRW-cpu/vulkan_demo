## 构建说明

### 前置要求

- CMake 3.25+
- Vulkan SDK
- GLFW
- GLM
- STB Image
- Assimp

### 构建步骤

1. 创建构建目录：
```bash
mkdir build
cd build
```

2. 配置项目：
```bash
cmake ..
```

3. 编译项目：
```bash
cmake --build .
```

4. 运行程序：
```bash
./vulkan-engine
```

## 代码说明

### 主要组件

- **Vulkan实例创建**: 初始化Vulkan运行时环境
- **物理设备选择**: 自动检测并选择合适的GPU
- **交换链管理**: 处理图像呈现和窗口表面
- **渲染管线**: 配置顶点着色、光栅化和片段着色
- **资源管理**: 顶点缓冲区、索引缓冲区、纹理和统一缓冲区
- **命令缓冲区**: 录制和执行渲染命令

### 模型加载

项目使用Assimp库加载3D模型，当前配置为加载FBX格式的鸭子模型。模型路径和纹理路径在代码中硬编码，可根据需要修改。

## 配置选项

在CMake配置中，可以通过以下选项控制构建：

- `VULKAN_BUILD_THIRDPARTY`: 控制是否构建第三方依赖（默认开启）

## 开发说明

### 调试支持

项目包含Vulkan验证层支持，在调试模式下会自动启用，帮助检测API使用错误。

### 窗口调整

支持窗口大小动态调整，交换链会在窗口大小变化时自动重建。

## 已知问题

- 模型和纹理路径为硬编码，需要根据实际文件位置调整
- 目前仅支持特定的模型格式
